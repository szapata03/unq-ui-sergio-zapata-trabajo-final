# Trabajo Final Integrador

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/szapata03/unq-ui-sergio-zapata-trabajo-final.git
```

Ingresar al directorio del proyecto:

```bash
cd unq-ui-sergio-zapata-trabajo-final
```

Instalar las dependencias:

```bash
npm install
```

## Ejecución

Iniciar el servidor de desarrollo:

```bash
npm run dev
```

Luego abrir el navegador en la dirección `http://localhost:5173`.
